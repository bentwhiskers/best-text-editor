## Installation
To install, 
- npm install (or just 'npm i') will be required for the following packages:
    > - Express v4.17.1
    > - Webpack v5.51.1
    > - webpack-dev-server v4.0.0
    > - webpack-pwa-manifest v4.3.0
    > - Babel 
    > - css-loader v6.2.0
    > - concurrently v5.2.0
    > - IndexedDB 
- These packages can be found in the package.json file(s). In the terminal, run: 
    > - npm i
